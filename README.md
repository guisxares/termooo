# 🟩 Clone do Term.ooo

Este é um clone do popular jogo de palavras **Termo**, inspirado no Wordle, desenvolvido com **HTML**, **CSS** e **JavaScript** puro.

## 📌 Sobre o Projeto

O jogo consiste em adivinhar uma palavra de 5 letras em até 6 tentativas. A cada palpite, as letras são coloridas para indicar se estão corretas e na posição certa.

✔️ Verde: letra correta na posição correta  
🟨 Amarelo: letra correta na posição errada  
⬜ Cinza: letra não existe na palavra  

## 🛠 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (vanilla)
